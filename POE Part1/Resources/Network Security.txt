What is Network Security?  
Network security refers to the policies, tools, and practices designed to protect a computer network from unauthorized access, cyberattacks, data breaches, and misuse. It ensures the confidentiality, integrity, and availability (CIA triad) of data transmitted across or stored within a network.

 Key Network Security Measures  

 1. Firewalls  
   - Acts as a barrier between trusted and untrusted networks.  
   - Types: **Packet-filtering, NGFW (Next-Gen Firewall), Cloud Firewall**.  

 2. Intrusion Detection & Prevention Systems (IDS/IPS)  
   - IDS monitors for suspicious activity.  
   - IPS actively blocks threats.  

 3. Virtual Private Network (VPN)  
   - Encrypts internet traffic to prevent eavesdropping.  

 4. Access Control  
   - Authentication (Passwords, MFA, Biometrics).  
   - Role-Based Access Control (RBAC) â€“ Limits user permissions.  

 5. Encryption  
   - Secures data in transit (SSL/TLS) and at rest (AES-256).  

 6. Antivirus & Anti-Malware  
   - Scans and removes malicious software.  

 7. Network Segmentation  
   - Divides networks into smaller zones to limit breach impact.  

 8. Patch Management  
   - Regularly updates software to fix vulnerabilities.  

 9. Email Security  
   - Blocks phishing, spam, and malicious attachments.  

 10. Wireless Security  
   - Secures Wi-Fi with WPA3 encryption, hides SSID, and uses MAC filtering.
